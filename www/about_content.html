What is Open Council?

Keep track of City Council and Committee meetings as they are published. The City's e-Agenda tool is great, but it requires that you check in to find out what's going on. Open Council monitors the agendas and minutes automatically and will tweet when new documents are found. So follow @LdnWatch and you'll never miss an update. You can browse all meetings too - and there is a handy dashboard for viewing meeting documents and agenda items.

Have a public discussion about Council and Committee meetings. Using the Comments tab in any meeting you can discuss the proceedings or agenda publicly. Please note that city officials are not monitoring the discussion. If you want to reach out to members of council, go here.

What's the tech behind Open Council?

OpenCouncil is an implementation (possibly also a mutilation) of Kevin O'Donnell's OttWatch system:

OttWatch runs on an Amazon EC2 "small" Unbuntu server, using Apache, PHP and Mysql. The GUI prettiness is courtesy of Bootstrap, and the REST style application design is aided by Epiphany. Source code is tracked in GIT and deployments are handled by Capistrano. All in all, a wonderful system to work in that couldn't be more different than my day job.

- Kevin O'Donnell
